# About PulseBoard

PulseBoard is a multi-game ranking platform built for gaming organizations that need transparent,
auditable, and real-time competitive standings across more than one title.

## Why We Built This

Most leaderboard tools assume a single game with a single score. That does not reflect how players
actually compete. Skilled players participate across multiple titles, and their reputation should
be visible across all of them — not buried in separate dashboards.

PulseBoard solves this by maintaining per-game scores alongside an overall cross-game ranking,
so a player who tops Apex Arena can still be compared fairly against a Turbo Drift specialist.

## Mission

Deliver the clearest possible picture of competitive performance — per game, per player, in real time.

## Principles

1. **Per-game accuracy over vanity aggregation.** Scores are always attributable to a specific title.
2. **Transparent ranking logic.** Higher score in a game = better rank in that game. No hidden weights.
3. **Full API parity.** Every page on this site has a JSON API equivalent for integrations.
4. **Operational simplicity.** Create, update, and delete player records without touching a database.
